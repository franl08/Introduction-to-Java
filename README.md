# CALC.java

Calculator for two integers

1. **How to use**
  - In the terminal:
    - javac Calc.java
    - java Calc *firstNum* *operation* *secNum*

2. **Possible operations:**
  - "+" (sum)
  - "-" (subtraction)
  - "*" (multiplication)
  - "/" (division)
  - "/i" (integer division)
  - "%" (remainder)

3. **Example**
  ```
  $ java Calc.java
  $ java Calc 5 % 2
  $ 5 % 2 = 1
