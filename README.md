# CALC.JAVA

Calculator for two integers

1. **How to use**
  - In the terminal:
    - java calc *firstNum* *operation* *secNum*

2. **Possible operations:**
  - "+" (sum)
  - "-" (subtraction)
  - "*" (multiplication)
  - "/" (division)
  - "%" (remainder)

3. **Example""
  > $java calc 5 % 2
  > $5 % 2 = 1
